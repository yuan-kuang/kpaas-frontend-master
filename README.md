# kplcloud 前端页面

## 拉取依赖

```
$ yarn install
```

## 工具

**使用yarn进包管理**

```
$ npm install yarn -g
```

## 设置镜像

```
$ yarn config set registry https://registry.npm.taobao.org
$ yarn config list
```



## 安装包

```
$ yarn install {包名}@{版本}
```

## 启动

```
$ yarn run
```

## Build

```
$ yarn build
```
